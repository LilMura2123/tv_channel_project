# 📺 TV Channel Web Application

A Django-based web application for managing TV channel content, including news and program schedules, with role-based access control and PostgreSQL integration.

---

# 🎯 Project Overview

This project simulates a real-world TV channel content management system.

The goal was to build a full-featured web application that allows:
- managing news and TV programs  
- handling user roles and permissions  
- analyzing basic platform statistics  

---

# 🧠 Key Features

- **User Authentication**
  - Registration and login system
  - Role-based access (Admin / Editor / User)

- **Content Management (CRUD)**
  - Create, edit, and delete news and programs
  - Different permissions depending on user role

- **Role-Based Access Control**
  - Admin → full access  
  - Editor → content management  
  - User → read-only  

- **Search & Filtering**
  - Search by title and content  
  - Filter and sort data  

- **Statistics Dashboard**
  - Number of users, articles, programs  
  - Visual insights into platform activity
